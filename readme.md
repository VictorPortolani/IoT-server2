# Projeto IoT - Monitor de Umidade do Solo / Soil Moisture Monitor

Este projeto consiste em um monitor de umidade de solo IoT integrado, no qual um Arduino lê os dados analógicos de um sensor (escala de 0 a 1023) e os transmite via porta serial (COM5, 9600 bps) para um servidor Python local baseado em Flask. O servidor trata esses dados em tempo real por meio de uma thread secundária dedicada — ou simula leituras dinâmicas entre 200 e 800 no modo de teste caso o hardware não esteja conectado —, envia a leitura periodicamente a cada 5 segundos para o banco de dados na nuvem Firebase Firestore na coleção "leituras" e disponibiliza uma API JSON `/api/dados` consumida por um painel web via JavaScript (`fetch`) a cada 10 segundos para exibir graficamente a umidade atualizada na tela sem necessidade de recarregar a página.

This IoT project is an integrated soil moisture monitor where an Arduino reads analog data from a physical sensor (scale from 0 to 1023) and transmits it via serial communication (COM5, 9600 bps) to a local Python Flask server. The server processes this data in real time using a dedicated background thread—or simulates mock readings between 200 and 800 for offline testing—periodically uploads the measurements every 5 seconds to a Firebase Firestore cloud database under the "leituras" collection, and provides a `/api/dados` JSON API which is polled by a web dashboard using JavaScript (`fetch`) every 10 seconds to display updated moisture values in real time without refreshing the page.
